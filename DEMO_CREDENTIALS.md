## ü§ñ AI Assistant Setup (Optional)

To enable the AI chat feature:

1. Get a Gemini API key from: https://aistudio.google.com/app/apikey
2. Set the environment variable in PowerShell:
   ```powershell
   $env:GEMINI_API_KEY = "your-api-key-here"
   ```
3. Restart the Flask app
4. Access the discharged portal to chat with AI

## ‚ö†Ô∏è Troubleshooting

### "Invalid Staff ID or password" error
- Make sure you ran `python init_db.py` to initialize the database
- Check the credentials above match exactly (case-sensitive for Staff ID)
- Try using the email if Staff ID doesn't work

### Patient portal not accessible
- Ensure the patient is marked as "discharged" (check database)
- Verify phone number is entered correctly (include dashes)
- Try the credentials shown in demo data

### Database locked error
- Close the Flask app and any other Python instances
- Wait a few seconds
- Run `python init_db.py` again

### AI chat not working
- Set `GEMINI_API_KEY` environment variable
- Restart the Flask application
- Check that the API key is valid
